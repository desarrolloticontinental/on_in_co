LogRead is a GUI utility to read, query, sort, merge and translate log 
files from all Progress components (e.g. AdminServer, AppServer, Messenger). 
LogRead is a 4GL application that runs only on the Progress Windows Client. 
For further documentation, please refer to LogRead's context-sensitive help.

INSTALLING LogRead

LogRead is provided in a zip file for supported versions of 
Progress (9.1D, 9.1E) and OpenEdge (10.0A, 10.0B). The zip file contains 
the following files:

- logread/README.TXT - this file
- logread/logread.pl - procedure library containing LogRead application
- logread/LogReadHelp.chm - LogRead help file

To install LogRead, extract the zip beneath your Progress working directory. 
Then add logread.pl to your PROPATH.

RUNNING LogRead

To run LogRead, run the program logread/logread.w, either with the 
-p parameter from the command line, or with the 4GL RUN statement.
